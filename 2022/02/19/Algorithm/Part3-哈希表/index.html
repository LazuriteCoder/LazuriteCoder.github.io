Part3 哈希表

Intro

哈希表利用特殊规则直接访问元素，可以做到将查询的时间复杂度将为O(1)，但是对应的需要开辟空间来进行存储，所以空间复杂度变成了O(1)。

Example

  1. 有效的字母异位词
  2. 两个数组的交集
  3. 快乐数
  4. 两数之和

1. 简单应用

对应有效的字母异位词，这里实际上是统计单词的字母出现次数，那么直接将其用哈希桶装起来就好了。 这里用了 Java 里的一个类似 foreach 的 for 循环操作。
